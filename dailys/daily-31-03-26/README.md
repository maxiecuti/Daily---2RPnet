## Evolução Técnica (Estudos & Labs)
Avancei na trilha de **Data Engineering** com foco em replicação e movimentação de grandes volumes de dados.

* **gcloud CLI:** Prática de comandos para gestão de recursos via terminal (Cloud Shell), otimizando o setup de projetos e datasets.
* **Moving Datasets:** Estudo de estratégias de migração de dados entre regiões e projetos no BigQuery, garantindo integridade e governança.
* **Datastream (CDC):** Início da implementação prática de **Change Data Capture**. Configurei a replicação de um banco **PostgreSQL** para o **BigQuery**, permitindo que os dados cheguem ao destino em tempo real conforme as atualizações ocorrem na origem.

---

## Inovação e Estratégia: "Data Guardian" (IA Proativa)
Refinei a proposta de transformar pipelines de dados tradicionais em sistemas inteligentes de **Observabilidade de Dados**.

* **Conceito:** Em vez de um fluxo passivo, a ideia é criar uma camada de inteligência que "interroga" o dado assim que ele é ingerido.
* **Funcionamento:** Utilização do **BigQuery ML** para detecção de anomalias estatísticas e integração com **IA Generativa (Gemini)** para traduzir falhas técnicas em insights de negócio.
* **Objetivo:** Notificar o cliente proativamente sobre desvios ou erros de integração antes mesmo que eles impactem os dashboards finais.
